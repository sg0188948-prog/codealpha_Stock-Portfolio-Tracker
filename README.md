# Stock-Portfolio-Tracker

A simple Python application to track your stock investments and portfolio performance.

## About Project
This project is made as part of **CodeAlpha Internship Program**.

## Features
- **Add Stocks**: Add new stocks with ticker, quantity and purchase price
- **Track Portfolio**: View total investment, current value and profit/loss
- **Live Price Fetch**: Get real-time stock prices using Yahoo Finance API
- **Portfolio Summary**: Shows total gain/loss % and overall portfolio value
- **Save Data**: Portfolio is saved in CSV file for future use
- **Easy to Use**: Simple command line interface

## Tech Stack
- **Language**: Python 3
- **Libraries**: `yfinance`, `pandas`, `tabulate`
